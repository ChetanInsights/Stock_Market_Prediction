# 📊 Stock Market Analysis & Prediction (Coal India)

## 🚀 Project Overview  
This project analyzes **5 years (2019–2024)** of stock market data for Coal India Limited to understand trends, extract insights, and build a foundation for stock price prediction.  

The workflow includes **data cleaning in Excel** and **interactive visualization using Power BI**.

---

## 🎯 Objectives  
- Analyze historical stock trends (Open, Close, High, Low, Volume)  
- Identify growth patterns and anomalies  
- Visualize insights using a Power BI dashboard  
- Prepare dataset for predictive modeling  

---

## 🛠️ Tools & Technologies  
- **Excel** → Data cleaning & preprocessing  
- **Power BI** → Dashboard & visualization  
- **GitHub** → Version control  

---

## 📈 Key Insights  

- 📌 **Strong Growth Trend:** Closing prices increased significantly from 2019 to 2024, showing long-term growth  
- 📌 **Peak in 2024:** Highest closing value recorded (~106K), indicating bullish momentum  
- 📌 **High Trading Volume:** 2021 had the highest volume (~3.8B), reflecting strong market activity  
- 📌 **Stable Price Range:** Small difference between average high (241.41) and low (234.88) suggests stability  
- 📌 **Post-COVID Recovery:** Noticeable rise in both price and volume after 2020  
- 📌 **Positive Sentiment:** Closing prices often remain close to or above opening prices  



---

## 🔍 Key Learnings  
- Clean data is essential for accurate analysis  
- Visualization simplifies complex data  
- Volume + price together provide deeper insights  
- Time-series analysis helps identify long-term trends
